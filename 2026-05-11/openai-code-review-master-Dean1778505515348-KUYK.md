
**严重性：🔴 高风险**

**问题分析：**
- 将公网可访问的内网穿透地址作为**默认值**写入代码仓库，存在严重安全隐患：该地址一旦提交到 Git 历史，即使后续删除，也可通过提交记录被他人访问，进而直接攻击后端服务。

**建议：**
